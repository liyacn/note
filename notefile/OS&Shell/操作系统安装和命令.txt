Windows创建软连接（需管理员身份运行）：（等价于unix的： ln -s 目标 链接）
CMD> mklink /D 链接 目标
	eg: mklink /D "%USERPROFILE%\OneDrive\Documents" "E:\Documents"
PS> New-Item -ItemType SymbolicLink -Path "链接" -Target "目标"
	eg: New-Item -ItemType SymbolicLink -Path "$HOME\OneDrive\Documents" -Target "E:\Documents"

Ubuntu桌面版24.04默认安装挂载2个分区
	EFI分区 /boot/efi FAT32
	根分区 / ext4
手动分区可选
	用户数据分区 /home ext4
	动态数据分区 /var ext4
	交换分区 swap swap (需要休眠才单独分配，1~2倍内存大小，系统默认使用交换文件更灵活)

查看定时任务：
crontab -l
编辑定时任务：
crontab -e
写入一行格式为： 分 时 日 月 周 命令
例如： 45 22 * * * cd /home/ubuntu/github/note;git pull
删除用户的crontab文件：
crontab -r
系统的crontab文件通常位于/etc/crontab，以root用户的身份执行任务。

apt是一个在Debian和Ubuntu中的Shell软件包管理器。apt命令执行需要超级管理员权限。
apt [options] [command] [package ...]
	options：-h（帮助），-y（当安装过程提示选择全部为"yes"），-q（不显示安装的过程）等等。
	command：要进行的操作。
	package：安装的包名。
apt常用命令
	列出所有可更新的软件清单命令：sudo apt update
	升级软件包：sudo apt upgrade
	可以将两个命令组合起来一键更新：sudo apt update && sudo apt upgrade -y
	列出可更新的软件包及版本信息：apt list --upgradeable
	升级软件包，升级前先删除需要更新软件包：sudo apt full-upgrade
	安装指定的软件命令：sudo apt install <package_name>
	安装一个软件包，但如果软件包已经存在，则不要升级它，可以使用 –no-upgrade
	如果只想升级，不要安装可以使用 --only-upgrade
	如果需要安装指定版本：sudo apt install <package_name>=<version_number>
	安装多个软件包：sudo apt install <package_1> <package_2> <package_3>
	更新指定的软件命令：sudo apt update <package_name>
	显示软件包具体信息：sudo apt show <package_name>
	删除软件包命令：sudo apt remove <package_name>
	清理不再使用的依赖和库文件: sudo apt autoremove
	移除软件包及配置文件: sudo apt purge <package_name>
	查找软件包命令： sudo apt search <keyword>
	列出所有已安装的包：apt list --installed
	列出所有已安装的包的版本信息：apt list --all-versions
桌面版系统可以通过：软件和更新>下载自>...选择一个源并保存，点击重新载入后会更新source.list
